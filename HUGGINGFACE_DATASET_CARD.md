# AI TRiSM Market Index Dataset

The definitive open-source reference dataset for the AI Trust, Risk, and Security Management (AI TRiSM) market.

## Dataset Description

- **Repository:** [alpha-one-index/ai-trism-index](https://github.com/alpha-one-index/ai-trism-index)
- **Dashboard:** [alpha-one-index.github.io/ai-trism-index](https://alpha-one-index.github.io/ai-trism-index/)
- **License:** CC BY 4.0 (data), MIT (code)
- **Version:** 1.0.0
- **Updated:** Weekly

## What's Included

| File | Description | Records |
|------|-------------|--------|
| `data/vendors.csv` | 60+ AI TRiSM vendor profiles with funding, capabilities, customers | 60+ |
| `data/market-sizing.json` | Market size estimates from 8 analyst firms | 8 |
| `data/regulatory.json` | Regulatory framework requirements and timelines | 15+ |
| `data/ma-activity.json` | M&A deals and funding rounds | 19 |
| `agents/vendors.csv` | 30+ agentic AI security vendors | 30+ |
| `agents/market-data.json` | Agentic AI security market sizing | 5+ |

## Coverage

- **60+ vendor profiles** across 10 AI TRiSM categories (Gartner four-layer framework)
- **Market sizing** from 8 analyst firms (~$2.95B in 2025, ~21.7% CAGR)
- **Regulatory landscape**: EU AI Act, NIST AI RMF, ISO/IEC 42001, GDPR, US state laws, China regulations
- **M&A activity**: 8 major acquisitions (2024-2025), 11 funding rounds
- **Agentic AI security**: OWASP Top 10 mapping, 30+ vendors, MCP security analysis
